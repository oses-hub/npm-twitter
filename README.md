<h1 align="center">npm-twitter</h1>

<div align="center">

npm-twitter Module Allows You To Get Twitter Download Link!



</div>

## Install

```sh
npm i npm-twitter
```

## Usage

```js
var twitter = require('npm-twitter')


                //id Of Tweet
twitter.download('1354113050639437824').then(url => {
    // Print The Url Link
    console.log(url)
})
```
